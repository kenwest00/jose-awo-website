# Design Brainstorm — José Awo Art Mockups

This project presents two distinct visual mockup concepts side-by-side. The landing page acts as a concept selector, then each concept shows multiple page designs (homepage, gallery, artwork detail, about, contact).

**Core Principles:** Warm off-white backgrounds, monospaced typography, material-driven animations, asymmetric layouts
**Color Philosophy:** Studio White (#F5F0EB) with Rusted Steel (#B7410E) accent — warmth of a physical studio
**Layout Paradigm:** Two-panel splits, asymmetric grids, horizontal scroll galleries
**Signature Elements:** SVG line-drawn signature, layered artwork frames at slight angles, fill-from-left hover effects

**Animation:** GSAP-driven scroll reveals, FLIP transitions, parallax at 80% speed
**Typography System:** Roboto Mono (headings) + Work Sans (body) — technical precision meets readability

## Selected Approach

Building BOTH Concept 1 (Industrial Brutalism) and Concept 2 (Cinematic Documentary) as side-by-side visual mockups. The user will see a concept selector landing page, then can explore each concept's full page designs with working animations and interactions.
